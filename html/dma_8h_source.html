29 /*-
30  * SCU DMA should be activated by the Master CPU
31  *
32  * - Keep in mind that:
33  * 1. DMA transfers does not begin until it is explictily started
34  * 2. DMA transfer level 2 will block the CPU during operation of level 1
35  *
36  * - SCU DMA is for transfers between different buses:
37  * 1. Work RAM-H <-> A-bus
38  * 2. Work RAM-H <-> B-bus
39  * 3. A-bus <-> B-bus
40  *
41  * - A-bus write prohibited
42  * - VDP2 area read prohibited
43  * - WORKRAM-L usage from SCU disabled
44  *
45  * - Restriction on write address addition value in DMA based on access
46  * address
47  *
48  * 1. WORKRAM-H -> 0x02 can be set
49  * 2. External areas 1 to 3 -> 0x02 can be set
50  * 3. External area 4 (A-bus I/O area) -> 0x00 and 0x02 can be set
51  * 4. VDP1, VDP2, SCSP -> all values can be set
52  */
