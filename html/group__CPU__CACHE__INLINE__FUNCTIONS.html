inlinestatic

Disable data replacement in the cache.

When data is fetched from memory, cache data is not written to the cache even if there is a cache miss.

static void cpu_cache_way_mode_set ( uint8_t  mode)
inlinestatic

Change the mode the cache operates.

The cache can be set to operate as a four-way set associative cache, or as a two-way associative cache and 2KiB RAM.

In the two-way mode, ways 0 and 1 are RAM.

modeWay mode.
