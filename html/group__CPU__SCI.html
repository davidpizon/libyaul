◆ cpu_sci_config_set()

void cpu_sci_config_set ( const cpu_sci_cfg_t *  cfg)

Configure a CPU-SCI for transfer.

Configuring the CPU-SCI in cfg does not start the transfer. To start the transfer, use either cpu_sci_enable_with_dmac (for SCI+DMAC mode) or cpu_sci_write_value_set (for normal mode). The CPU-SCI is forcefully stopped upon starting the configuration. If the CPU_SCI is currently operating int SCI+DMAC mode, use cpu_dma_wait with the corresponding DMA channel to wait until the transfer is complete.

Parameters
[in]cfgThe CPU-SCI transfer configuration.
