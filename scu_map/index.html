Read and write¶

MEMORY_READ(t, x)¶

Read value of size t from address x.

MEMORY_WRITE(t, x, y)¶

Write value y to address x of size t.

MEMORY_WRITE_AND(t, x, y)¶

Bitwise AND value y to address x of size t.

MEMORY_WRITE_OR(t, x, y)¶

Bitwise OR value y to address x of size t.
